# momento

Momento is a mobile application designed with a dual light and dark theme interface. The final design screens showcase a clean onboarding flow, intuitive login, and a modern dashboard with interactive elements. The app emphasizes user-centric design, smooth navigation, and consistent visual hierarchy. Each screen highlights clickable sections and modular layouts, ensuring both functionality and aesthetic appeal. Built with Flutter, the app leverages Provider for state management and Flutter Keys for precise widget rebuilding, delivering a seamless and responsive user experience.
